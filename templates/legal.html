LEGAL

MSPA To Go is a project made by an independent developer, not associated with Andrew Hussie, MS Paint Adventures, What Pumpkin, VIZ Media, or the Homestuck Independent Creative Union (HICU). It acts as a proxy to the official MS Paint Adventures (www.mspaintadventures.com), requesting assets from MS Paint Adventures, transforming them as needed, then sending them to the user. Only a small portion of MSPA comic content is stored permanently on the server. This includes:

  • Transcriptions of certain chat logs that were originally images for a better reading experience on mobile devices
  • Text used for the banner tooltips in the Scratch interlude

Some other assets taken from MSPA are also stored on the server. These include:

  • Images and audio used for the pageframe (some of which are modified).
  • The JavaScript code and images used for the fireflies from pages 009000 to 009024 and pages 009289 to 009303.

When the server is in Offline Mode, any assets normally requested from MS Paint Adventures are downloaded by the server operator and stored on the server. To see a comprehensive list of these assets, see scripts/dump.js in the project's source code.

The main instance of MSPA To Go (mspa.chadthundercock.com) does not run in Offline Mode.
