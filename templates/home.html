MSPA TO GO

MSPA To Go is an application server that serves pages from the MS Paint Adventures server in a format that is digestible on desktop computers and mobile devices without any externa downloads since MS Paint Adventures site no longer works, and Homestuck.com is incredibly broken.

If you want to use the Homestuck.com page numbers, desktop mode, a different theme or font, and more, be sure to check out the Options page!

Also, if you want Homestuck.com and MSPaintAdventures.com to redirect here, be sure to check out this userscript!

MS Paint Adventures is the work of Andrew Hussie, and this website's operator nor the author of the project (aubymori) are associated with MS Paint Adventures, What Pumpkin, VIZ Media, or the Homestuck Independent Creative Union. MSPA To Go is made out of love for MSPA and a desire for it to be accessible to those using mobile devices.
