PRIVACY

The MSPA To Go software itself does not collect any user data, user options are stored only on the client in cookies, and temporarily on the server when making requests.

The main MSPA To Go instance (mspa.chadthundercock.com), however, does log information about requests made to the server, including:

  • When the request was made, and how long it took
  • The client's public IP address
  • The URL requested by the client
  • HTTP headers, including the User Agent
  • Any request body sent by the client
